##Installation (Quick and Simple!)

### 1. First you need the gcc compiler (If you are in GNU/Linux you already have one)
- You can download it from: https://gcc.gnu.org/

### 2. Compile it

	gcc hangman.c -o hangman.exe 

### 3. Enjoy!
